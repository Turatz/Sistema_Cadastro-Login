🔐 Sistema de Cadastro e Login

Um sistema web completo com frontend + backend, desenvolvido em Django, que permite cadastro, autenticação e gerenciamento de usuários.

📌 Sobre o Projeto

Este projeto é uma aplicação web fullstack utilizando o framework Django, com banco de dados SQLite e interface frontend integrada via templates.

O sistema permite que usuários se cadastrem, façam login e logout.

🚀 Funcionalidades
✅ Cadastro de usuários
🔑 Login e autenticação
🚪 Logout
💻 Interface web (frontend) integrada
